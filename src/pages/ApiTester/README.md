# ApiTester Folder

Thư mục này chứa các thành phần liên quan đến việc kiểm thử và chạy thử các API.

## Các file chính:
- `TestApi.jsx`: Giao diện kiểm thử API (giống Postman), cho phép gửi các request GET, POST, PUT, DELETE và xem kết quả trả về.
- `ApiTesterPage.jsx`: Trang bao quát cho các công cụ kiểm thử.
