# Dashboard Folder

Thư mục này chứa Layout chính và các trang quản trị hệ thống, cá nhân.

## Các file chính:
- `DashboardLayout.jsx`: Khung sườn của Dashboard (Sidebar + Header).
- `AdminOverview.jsx`: Thống kê tổng quan cho Admin.
- `UserManagement.jsx`: Quản lý danh sách người dùng.
- `RevenueManagement.jsx`: Quản lý doanh thu hệ thống.
- `Profile.jsx` & `EditProfile.jsx`: Quản lý thông tin cá nhân người dùng.
- `Settings.jsx`: Cài đặt hệ thống (Giao diện, bảo mật).
- `History.jsx`: Nhật ký hoạt động.
